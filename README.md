# 🌐 タイルエクスポート GDAL

このリポジトリは、**GDAL** を使用して地理空間データから画像タイルを出力する Python スクリプトです。

## 📁 ファイル構成

| ファイル名         | 説明                                 |
|-------------------|--------------------------------------|
| `main.py`         | メインの処理スクリプト              |
| `user_input.ini`  | タイル出力に必要なパラメータ設定ファイル |

## ⚙️ 使用方法

1. **`user_input.ini` を編集：**
   タイル化したいGeoTIFFや出力形式などを記述します。

2. **以下のコマンドで実行：**
   ```bash
   python main.py
